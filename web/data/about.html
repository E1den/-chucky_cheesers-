In this project, we developed a web application, where people can add photos, texts, and videos to stories. The web application is mobile responsive and compatible. The user should be able to continue adding content to one of the existing comics or start a new comic. The user interface resembles a comic book. In order to create semantically meaningful stories, the person who has created the comic can add tags so others with similar interests may find their comic.
